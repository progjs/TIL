# vi editor 명령어

- vi 에서 편집 중인 것을 저장하지 않고 빠져나왔을 때

  ```
  $ rm -rf swap 파일명
  ```



# linux ubuntu terminal 명령어

- root 계정 => 다른 계정

  ```
  ~# su - 계정명
  ```

- root 계정으로 변경

  ``` 
  ~$ sudo su
  비밀번호 입력
  ```
